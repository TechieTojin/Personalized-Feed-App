## Getting Started

### Prerequisites

* Node.js 12.3.1
* MongoDB
* Auth0 Account

### Installation

1. Clone the repository
2. Install dependencies
   ```
   npm install
   ```
3. Set up environment variables
4. Run the application
   ```
   npm start
   ```
